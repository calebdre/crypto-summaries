Nebulas is a value-based blockchain operating system and search engine..  It features Nebulas Rank, which measures value by considering liquidity and propagation of the address, Nebulas Force (NF), which supports upgrading core protocols and smart contracts on the chains, and Developer Incentive Protocol (DIP), designed to build the blockchain ecosystem in a better way..  NAS is an Ethereum-based ERC20 token that serves as currency on Nebulas ecosystem.
