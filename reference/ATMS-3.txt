The Advanced File Index (“AFIX”) combines the advantages both the IPFS and the Novusphere blockchain-based database, using the latter as a database layer for the content content on the IPFS p2p network..  This database allows Novus to build an advanced index for any files that exist in the IPFS network and provides metadata for each file..  Atmos, a Proof of Stake cryptocurrency, is the native value token in the Novusphere blockchain and will be used to pay for content rating, content request, dispute resolution and gateway advertisement of the Novusphere AFIX.
